# NIPS Konferans Makaleleri (1987-2015) - Dönem Sınıflandırması

Bu proje, Gazi Üniversitesi Bilgisayar Mühendisliği Veri Bilimine Giriş dersi kapsamında geliştirilmiştir.

Çalışmada, 1987-2015 yılları arasındaki NIPS makalelerinin kelime frekans matrisleri (11.463 öznitelik) kullanılarak, makine öğrenmesi algoritmalarıyla "Klasik" ve "Modern" dönem sınıflandırması yapılmıştır.

**Kullanılan Algoritmalar:** YSA, Rastgele Orman, SVM, Lojistik Regresyon, Basit Bayes, Karar Ağacı, KNN.
**En Yüksek Başarı:** YSA (%94.75)

Projenin Python (Google Colab) kodlarına ve detaylı araştırma raporuna (PDF) bu depo üzerinden erişebilirsiniz.
